# Screen 后台运行管理脚本

此脚本能够更好的帮助你管理screen后台下的任务，做到管理简单化

## 下载并部署

使用以下命令即可

```shell
wget -N https://cdn.jsdelivr.net/gh/Misaka-blog/screenManager@master/screen.sh && chmod -R 777 screen.sh && bash screen.sh
```

快捷方式 `bash screen.sh`
